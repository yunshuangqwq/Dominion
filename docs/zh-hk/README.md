## 前言

首先十分感謝您選擇並使用本插件。

Dominion 是一個完全開源、免費，專為高版本開發，面向未來的全新領地保護插件。插件目前處於快速的開發、叠代階段，我們會盡力保證插件的穩定性和功能性。

如果您在使用過程中遇到任何問題，或者有任何建議，歡迎在 [GitHub ISSUE](https://github.com/ColdeZhang/Dominion/issues)
提交您的問題或想法。
